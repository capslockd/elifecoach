ELife Coach Website

A modern, responsive landing page for ELife Coach, featuring Pastor Jimm & Pynqi Gorgonia.
Designed to present coaching services, build trust, and drive users toward booking a discovery call.

------------------------------------------------------------

OVERVIEW

This is a single-page HTML website built with:
- Vanilla HTML, CSS, and JavaScript
- Fully responsive layout (desktop to mobile)
- Smooth scrolling navigation
- Clean, faith-centered design aesthetic

The page is structured to guide visitors through:
Awareness -> Trust -> Services -> Process -> Action

------------------------------------------------------------

FILE STRUCTURE

/project-root
  index.html   (Main website file)
  README.txt   (Project documentation)

------------------------------------------------------------

SECTIONS BREAKDOWN

1. Navigation
- Fixed header with anchor links
- Smooth scrolling
- Call-to-action button: Book a Call

2. Hero Section
- Main headline and value proposition
- Two-column layout (text + image)
- Primary conversion buttons

3. Stats Bar
- Social proof indicators:
  Lives transformed
  Years in ministry
  Coaching programs
  Faith-centered focus

4. About Section
- Introduction to Jimm & Pynqi
- Mission and coaching philosophy
- CTA to start journey

5. Services Section
- Life Coaching
- Couples Coaching
- Leadership Coaching

6. Approach Section
- 4-step framework:
  1. Discovery
  2. Roadmap
  3. Accountability
  4. Breakthrough

7. Call-To-Action
- Contact form (name, email, phone)
- Uses mailto to open user's email client

8. Footer
- Quick links
- Contact details
- Branding

------------------------------------------------------------

FEATURES

Responsive Design:
- Desktop: grid layout
- Mobile: stacked layout
- Prevents horizontal overflow

Smooth Scrolling:
Enabled via CSS scroll-behavior

Scroll Reveal Animation:
Uses IntersectionObserver in JavaScript

Email Integration:
Uses mailto link to send form details
